## 01 / Profile

I'm a first-year Information Technology student at **Posts and Telecommunications Institute of Technology — HCMC Campus (PTIT HCM)**.

Most of my work starts with a practical problem: a slow workflow, repetitive task, messy dataset, backend requirement, or system that needs to be easier to operate. I usually build the first working version in **Python**, inspect the logs and data flow, then improve the structure and performance from there.

My long-term direction is **software optimization and system performance**. I am also interested in machine learning when it solves a concrete engineering problem — particularly **intrusion detection, information retrieval, and multimodal video search**.

## 05 / Selected experiments

### Information Retrieval

Worked with `TF-IDF` and `BM25` to understand ranking, scoring and the difference between keyword retrieval and semantic retrieval.

### Network Traffic Classification

Experimented with feature selection and model evaluation on CICIDS2017. The goal was not only classification accuracy, but also reducing the feature set and fitting the model into a real detection pipeline.

### Video Search Systems

Currently studying how semantic features, object metadata, speech transcripts and temporal signals can be narrowed through a retrieval funnel before expensive reasoning is used.

### Backend & Deployment

Built FastAPI services backed by PostgreSQL and deployed Python applications on Ubuntu with Nginx, Uvicorn and systemd. Most of the useful lessons came from debugging configuration, process and database failures in real deployments.
